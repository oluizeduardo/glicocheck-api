## ðŸš€ How to run locally
- Clone this project.
    - `git clone https://github.com/oluizeduardo/glicocheck-api.git`
- Access the root folder.
    - `cd glicocheck-api`
- Configure the environment variables.
    - `cp .env-example .env`
- Install all the dependencies.
    - `npm install`
- Run the server.
    - `npm run dev`
